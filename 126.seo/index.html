织梦后台登录页面去广告

DEDECMS 5.7 登录页面的几个广告去法

打开后台，如图

织梦后台登录页面去广告,编辑/dede/login.php文件

 

//更新服务器
require_once (DEDEDATA.‘/admin/config_update.php’);

if ($dopost==‘showad’)
{
    include(‘templets/login_ad.htm’);
    exit;
}



修改为

//更新服务器 require_once (DEDEDATA.'/admin/config_update.php'); 

if ($dopost==‘showad’)
{
    // include(‘templets/login_ad.htm’);
    exit;
}

就是在前面加 // ，注释掉这一句include(‘templets/login_ad.htm’);

再看效果~
