谷歌浏览器正常跳转扩展 Google SSL Privacy Real Link Chrome Firefox

最近用google甚是蛋疼，要么中断，要么跳转几分钟也打不开!

Google SSL Privacy Real Link 是款开源的浏览器扩展，支持 Firefox 与 Chrome，用来去除 Google 搜索结果页面的跳转过程，从而使搜索结果直接达到页面。安装 Google SSL Privacy Real Link 后就可以打开搜索结果并正常跳转了。
官方主页 http://hi.baidu.com/strikewelkin/item/cfe67d0e56bcdfcf75cd3cff
百度盘http://pan.baidu.com/share/link?shareid=23175&uk=973433708

注意此脚本仅仅是省略了跳转过程，无任何其他功能。作者并未将 Chrome 扩展发布在应用商店（想发布应用需要5美金入场费…），只需下载 crx 扩展文件后，打开 Chrome 的扩展页面（chrome://extensions/），将 crx 文件拖入就可以正常安装了。
