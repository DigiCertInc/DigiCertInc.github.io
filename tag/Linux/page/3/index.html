Linux下提高效率的命令与快捷键

一、bash中的快捷键

光标移动命令

Ctrl + a - 跳到命令行行首
Ctrl + e - 跳到命令行尾
Ctrl + b - 回跳一个字符
Ctrl + f -往前跳一个字符

字符删除快捷键

Ctrl + d - 删除光标所在字符
Ctrl + k - 从光标处删除到行未尾
Ctrl + u - 从光标处删除到行首
Ctrl + w - 删除最后输入的单词
