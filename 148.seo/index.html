推荐十二款免费的Chrome应用

2011-10-22 Updated:2016-08-20

Contents

如果你正在使用Chrome浏览器，那么你一定要到Chrome Web Store里看一看。Chrome不仅仅是一款浏览器，它还支持很多应用程序。科技网站PCWorld近日就为大家精选了12款Chrome应用。

如果你正在使用Chrome浏览器，那么你一定要到Chrome Web Store里看一看。Chrome不仅仅是一款浏览器，它还支持很多应用程序。科技网站PCWorld近日就为大家精选了12款Chrome应用：

愤怒的小鸟

《愤怒的小鸟》是一款风靡全世界的移动游戏。这款Chrome应用将同样的游戏体验移植到浏览器中，绝对是打发时间的最佳选择。

《愤怒的小鸟》安装：

https://chrome.google.com/webstore/detail/aknpkdffaafgjchaibgeefbgmgeghloj

Kindle Cloud Reader

在电脑上阅读一整部书会很疲倦，这就是电子书阅读器发明的原因。当然，Kindle Chrome应用并不是一款专为整部书准备的应用，当用户不使用电子书阅读器，它就可以填补空白。

Kindle Cloud Reader安装：

https://chrome.google.com/webstore/detail/icdipabjmbhpdkjaihfjoikhjjeneebd

Aviary Audio Editor

与Pixlr图片编辑应用相似，Aviary Audio Editor是一款基于网页的音频编辑应用。Aviary在网页中提供一个多轨音频编辑，其功能性与苹果的Garage Band相当。

Aviary Audio Editor安装：

https://chrome.google.com/webstore/detail/ajiijeebjcmkhdplmollbjpljcnelfhn

Zombie Drop

Zombie Drop是一款物理解谜类游戏。同样，它也非常适合短暂的休闲。

Zombie Drop安装：

https://chrome.google.com/webstore/detail/mhmhllfgcoopjdmcmdeobhgimokcabmc

Feedly

Feedly是一款RSS订阅源管理应用。阅读器应用有很多，但Feedly独以可读性和设计见长。Feedly可以同步Google阅读器参数，按热度和最近条目排序，然后图文格式排版以便用户浏览。

Feedly安装：

https://chrome.google.com/webstore/detail/hipbfijinpcgfogaopmgehiegacbhmob?hc=search&hcp=main

Planetarium

对于天文爱好者而言，Planetarium是一款非常实用的行星和星座定位应用，其界面也非常出色。

Planetarium安装：

https://chrome.google.com/webstore/detail/gheikhdfflhlbemfmhcfpeblehemeklp

Word²

Word²是一款多人拼字游戏应用，与Scrabble非常相像。用户不仅可以与好友一起游戏，而且可以与其他人进行拼字比拼。

Word²安装：

https://chrome.google.com/webstore/detail/lpibnckjjeaabeepofhfmmpjmnomohee

Gojee

Gojee可以帮助用户搜索需要的食谱。Gojee的用户界面非常有特色：Gojee以大图片提供食物的图片。

Gojee安装：

https://chrome.google.com/webstore/detail/ajebcmdcgoggdncokkbdifohckmfpgnb

Hipmunk

如果你经常在各地奔波，你可能会需要Hipmunk。Hipmunk是一款航班对比应用，用户可对比不同航班的价格，距离和旅行时间。

Hipmunk安装：

https://chrome.google.com/webstore/detail/jeabbdefhlelidlhahnfpbllaomkioke

Pandora

Pandora是一款知名流音乐服务。在Pandora网络应用中，用户输入歌曲名或艺术家，Pandora便可根据用户的选择定制一个播放列表。

Pandora安装：

https://chrome.google.com/webstore/detail/fbangkleohkafngihneedemihgfeikcl

Sinuous

在Sinuous中，用户需要移动鼠标躲避飞行的红点。游戏简单，易于上手，失败后玩家可以立即重新游戏。

Sinuous安装：

https://chrome.google.com/webstore/detail/omlmnomieeknagejjojcpdomnbnbchdl

Netflix

Netflix的即时观赏服务是最好的娱乐平台之一。Netflix Chrome还兼任Chrome OS，Chrome用户现在可以直接观赏Netflix即时视频。

Netflix安装：

https://chrome.google.com/webstore/detail/deceagebecbceejblnlcjooeohmmeldh
