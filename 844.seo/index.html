Warning scandir() has been disabled for security reasons in的解决方法

博客搬家后发现不能编辑主题了，错误如下：

Warning: scandir() has been disabled for security reasons in

原因是php禁用了scandir函数导致，我们去修改php.ini配置文件

在disable——functions后面 去掉scandir，保存后去把LNMP reload下 
 ./lnmp reload


就ok了。
